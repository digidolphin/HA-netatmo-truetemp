# Netatmo TrueTemp Home Assistant Add-on

Unofficial Home Assistant add-on that exposes Netatmo “True Temperature”
control via a local HTTP API.

Built on `py-netatmo-truetemp-cli`. (https://github.com/py-netatmo-unofficial/py-netatmo-truetemp-cli)

## What it does
- Lists Netatmo thermostat rooms
- Sets “True Temperature” for a room
- Intended to be used from Node-RED or Home Assistant automations

## Status
Early / experimental.  
Not affiliated with Netatmo.
